# DNS 查询优化

### 一、 缓存

> 缓存是双刃剑，提高了 DNS 查询的速度，也会影响 DNS 在 IP 变更时不能及时更新到最新。

DNS 查找的各个环节会对查找结果做缓存，从而提高下次查询的速度。
